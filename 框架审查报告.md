## 五、风险提示

> ⚠️ **关于 Spring Security**
> 
> 当前 pom.xml 引入了 `spring-boot-starter-security`，但**没有任何配置**。
> 这意味着：
> - 项目启动后，**所有 HTTP 接口都会被保护**
> - 默认用户名：`user`，密码：控制台打印的 UUID
> - 没有 `/login` 页面（因为没写前端）
> - **如果直接调用 API 会返回 401/403**

**建议：** 如果你们暂时不需要复杂的安全体系，可以先**移除这个依赖**，后面再加回来并配置 JWT。或者我现在就帮你配好一套基于 JWT + Spring Security 的认证体系。
